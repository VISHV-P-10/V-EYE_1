V-EYE – AI Bias Detection & Correction Toolkit

Detect • Fix • Explain bias in AI systems — all in one place.

🚨 Problem

AI systems often inherit bias from real-world data, leading to unfair decisions in:

🏦 Loan approvals
💼 Hiring systems
🏥 Healthcare predictions

Most existing tools are complex and not beginner-friendly.

💡 Solution

FairLens is an interactive web app that allows users to:

Upload datasets
Detect bias across sensitive groups
Visualize disparities
Fix bias with one click
Understand why bias occurs using AI

👉 Think of it as “Grammarly for AI fairness”

✨ Features
📂 CSV Dataset Upload
🎯 Target & Sensitive Column Selection
📊 Bias Detection (Demographic Parity)
📉 Visualization Dashboard
🛠️ One-click Bias Mitigation
🔄 Before vs After Comparison
🤖 AI-based Bias Explanation
🎨 Premium UI with custom loader
⚙️ Tech Stack
Python
Streamlit
Pandas
Matplotlib
Scikit-learn
🧠 How It Works

🚀 Getting Started
1. Clone the repo
git clone https://github.com/your-username/fairlens.git
cd fairlens
2. Install dependencies
pip install -r requirements.txt
3. Run the app
streamlit run app.py
